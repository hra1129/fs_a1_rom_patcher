MSX2 Panasonic FS-A1 の 1Mbit MaskROM を取り外し、
代わりに 256Kbit EP-ROM を 4つ取り付けるボートです。

MAIN-ROM
SUB-ROM + 16KB-ROM
DESKPACK1
DESKPACK2

この4つに分かれているので、
必要に応じて未装着や別のROMに書き替えるなどして
使うことが出来ます。

KiCad_project/fs-a1_rom_patch_rev3.zip をダウンロードし、
JLCPCB の注文フォームで指定すれば、基板を購入することが
出来ます。
